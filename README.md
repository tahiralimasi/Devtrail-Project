# Smart Interview Assistant

## Overview

Smart Interview Assistant helps users prepare for interviews by generating questions from uploaded resumes and evaluating answers.

## Features

* Resume upload
* Skill extraction
* AI question generation
* Answer evaluation
* Score and improvement tips

## Tech Stack

* Python
* FastAPI
* OpenAI API
* HTML
* CSS
* JavaScript

## Workflow

1. Upload resume
2. Extract resume text
3. Generate interview questions
4. Submit answers
5. Get score and feedback
